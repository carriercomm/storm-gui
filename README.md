storm-gui
=========

wxpython interface to stormssh.

### installation ###

<a href="http://www.wxpython.org/download.php#stable">install wxpython</a> for your platform first. after that:

```
$ sudo pip install storm-gui
```
or if you like 90s:

```
$ sudo easy_install storm-gui
```


### running ###

just type:

```
$ storm-gui
```
